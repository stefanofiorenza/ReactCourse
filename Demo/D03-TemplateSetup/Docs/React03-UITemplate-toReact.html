- Step 3.4 : Fix errors HTML to XHTML

HTML Comments
Check for

  • html comments
<!-- .. -->

If you want to keep them at all, should be changed with

{/* .. */}

HTML unclosed tag

  • input
  • br
  • hr
  • img

Should be replaced by XHTML compliant tags:

<input .. />
<br/>
<hr/>
<img .. />

- Step 3.5 : Fix errors XHTML to JSX
Check complaints in browser CTRL+I

Javascript function handlders

Remove javascript function calls:

href="javascript:void(0)"

Should be replaced by:

href=""

JSX rules replacements:

  • class -> className
  • for -> htmlFor
  • tabindex ->tabIndex
  • checked -> defaultChecked={true}
  • style ->style={styleJsObject}
